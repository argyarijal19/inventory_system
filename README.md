Cara menjalankan backend nya adalah

1. setelah di clone maka jalankan perintah
   python -m virtualenv venv
2. kemudian jalankan
   source venv/Scripts/activate
3. kemudian install dependencies
   pip install -r requirements.txt
4. jalankan backend
   python main.py
